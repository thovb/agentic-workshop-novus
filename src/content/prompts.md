# Spec Kit Prompts

Use these prompts with Spec Kit and your preferred AI assistant.

## General workshop prompt

```text
Create a workshop plan that guides me through building a simple website with Vite, React, Tailwind CSS, and Spec Kit.
Include a checklist of the prerequsites, setup commands, and a review section.
```

## Spec Kit initialization prompt

```text
Initialize Spec Kit for my AI workflow and create the configuration files needed to use `specify` with GitHub Copilot.
Include the exact commands for `uv` and `specify`.
```

## Prompt template for task generation

```text
Given the current repository files and the workshop requirements, generate the next highest-value task for the project.
Return only the task description and the exact command I should run next.
```
