## Notes

`Impeccable` is a design-quality CLI tool for detecting UI anti-patterns, not a UI component framework. This starter uses Tailwind CSS for the UI design, with a lightweight custom theme built on top.

## New site structure

- `Home` — overview of the workshop website
- `Prerequisite` — renders `agentic_workshop_setup.md` as a setup guide
- `Prompts` — stores Speckit prompt content in `src/content/prompts.md`
